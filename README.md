# KodTrain LMS

KodTrain is a modern frontend Learning Management System designed for corporate employee training.

## Features

* Interactive training modules
* Lesson and quiz flow
* Progress tracking dashboard
* Leaderboard system
* Achievement badges

## Modules Included

* Workplace Safety
* Cybersecurity Awareness
* Communication Skills
* Project Management
* Customer Service

## Tech Stack

* TypeScript
* Vite
* HTML / CSS

## Running Locally

```
npm install
npm run dev
```

## Deployment

This project can be deployed easily using Vercel or Netlify.
